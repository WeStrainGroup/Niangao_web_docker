Welcome to Niangao!

Niangao is an all-in-one tool for processing Sanger sequencing results (.ab1 files).

  • Extracts continuous high-quality sequences in FASTA format from your .ab1 files, enabling downstream analysis.

Assembly & BLAST

  • Assembles paired "_Fwd.ab1" and "_Rev.ab1" files into contigs using CAP3.
  • Provides either the successfully assembled contig or the longest clean input (Fwd or Rev) sequence.
  • Allows you to BLAST your sequences, choose the database, and set the number of hits per page.
